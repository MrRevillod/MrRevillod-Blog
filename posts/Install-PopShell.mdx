## Instalación usando yay 

### Versión normal
<CodeBox text='yay -S gnome-shell-extension-pop-shell' />

### Versión precompilada:
<CodeBox text='yay -S gnome-shell-extension-pop-shell-bin' />

### Version repositorio git:
<CodeBox text='yay -S gnome-shell-extension-pop-shell-git' />
